# SUDOKU SOLVER

A graphical Sudoku Solver in Java to solve and visualize a Sudoku game

## 🚀 Running Sudoku:
To run Sudoku on the file Sudoku 

```sh
javac Sudoku.java
java Sudoku 10
```
Note: 10 is the number of given initial values on the broad
